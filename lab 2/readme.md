Description of HTML Portfolio Code:
This project is a basic HTML Portfolio Website created using simple HTML tags.
It contains different sections like:
  1.Heading – To display the heading of the file.
  2.About Me Section – A short introduction with an image.
  3.My Projects Section – A list of projects with one-line descriptions.
  4.Skills - To display level of knowladge of html,css and javascript.
  5.Contact Information – Email or phone number.
The code is written in pure HTML, which makes it easy to understand for beginners.
It uses basic tags like:
    1.<h1>, <h2>, <h3> → For headings
    2.<p> → For paragraphs
    3.<img> → For adding images
    4.<a> → For clickable links
    5.<br> → For line breaks
Description of CSS Portfolio Code:
1.Basic CSS used to make layout simple and clean.
2.Text aligned to center for better presentation.
3.Blue color theme applied to headings, borders, and buttons.
4.Skills and contact section shown using a styled table.
5.“Back to Top” button placed at bottom-right using fixed position.
6.Input fields and textarea styled for neat look.
7.Simple spacing added using margin and padding.
